Installation

1. Copy hook to your project

Terminal 
cp -r hooks/{{ page.slug }}/ /your-project/.claude/hooks/{{ page.slug }}/

2. Add to settings.json

.claude/settings.json 
{
  "hooks": {
    "{{ page.event_type }}": [{
      "matcher": "{{ page.matcher | default: 'Bash' }}",
      "hooks": [{
        "type": "command",
        "command": "$CLAUDE_PROJECT_DIR/.claude/hooks/{{ page.slug }}/hook.sh"
      }]
    }]
  }
}

3. Make executable

Terminal 
chmod +x .claude/hooks/{{ page.slug }}/hook.sh
