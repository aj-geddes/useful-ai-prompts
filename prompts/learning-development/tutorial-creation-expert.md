## Description

This prompt combines expert tutorial creation methodology with instructional design specialization to develop comprehensive, engaging tutorials that guide learners through practical skill acquisition with clear progression and measurable outcomes. It employs proven instructional design principles, learning psychology, and multimedia integration to create tutorials that maximize learning effectiveness and learner satisfaction.

      const email = document.getElementById("email").value;
      const message = document.getElementById("message").value;
